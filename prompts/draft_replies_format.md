# Draft Replies — Display Format

## Post Card Fields
- Checkbox (replied/not)
- Post title
- Author name
- Author bio summary (1-2 sentence AI-generated context)
- Topic badge
- Suggested repliers
- Skool URL link
- Post date
- Thread context: comment count + commenter names (up to 6, then "+X more")
- Post summary (1-2 sentence AI-generated)
- Full post content (expandable)
- Draft reply (editable)
- Separator between posts

## Bio Summary Generation
Synthesize from raw bio, location, and activity data.
Example: "Runs education companies in Miami. Mostly asking questions about scaling — likely in growth phase."

## Post Summary Generation
- Seeker: State what they're asking and the core constraint
- Giver: State what they're sharing and the key result
- Conversational: State the vibe/topic in plain language
